## 信源使用策略

### 日常跟踪

- **每周必查**: DORA博客、GitHub博客、Pragmatic Engineer、思码逸
- **每月必查**: 各大厂技术博客、QECon、知识星球动态
- **按需深挖**: 学术论文、标准文档、大会录播

### 深度调研

- **框架研究**: DORA报告 + 学术论文 + 官方文档
- **案例研究**: 大厂技术博客 + 大会演讲 + 公众号深度文章
- **趋势判断**: Gartner + ThoughtWorks Radar + 年度报告

### 优先级动态调整

| 信号 | 动作 |
|------|------|
| 新年度报告发布 | 第一时间学习、纳入知识库 |
| 新Newsletter/博客发现 | 试读2-3期，评估后纳入 |
| 信源质量下降 | 降级或移出清单 |
| 大会/活动日历 | 提前关注、记录Session |
