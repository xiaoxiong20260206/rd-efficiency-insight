# rd-efficiency-Insight

研发效能洞察平台 -- 林克的研发效能领域深度追踪与知识沉淀项目。

## 项目定位

对标 `AI-Insight/`，专注于研发效能领域的：
- 深度调研与趋势分析
- 知识体系建设与沉淀
- 企业/人物/信源追踪
- 洞察报告输出

## 技能关联

- 主技能：`rd-efficiency-insight` -- 研发效能洞察平台专属技能
